Transformar datos Item Volumen

Esta sección permite transformar los datos crudos de Item Volumen en un archivo de insumo para oasis. Para realizar este proceso el Item Volumen se deben cumplir los siguientes requisitos

Estructura Nombre Archivo
"pais"_"idfuente"_"periodicidad"_"categoria"_"ANOMES".xlsx
Estructura Archivo
Es un archivo de Item Volumen en formato excel con 2 hojas (WSP_TOC, WSP_Sheet1)
Columnas del archivo
Se espera que la hoja "WSP_Sheet1" tenga las siguiente columnas: ["MERCADO" "VARIABLE" "SEGMENTO" "SUB-SEGMENTO-1" "FABRICANTE" "MARCA" "PRESENTACION-11" "EMPAQUE 104" "PESO 104" "ITEM" "DESCRIPCIÓN LARGA" "TAG"] más las columnas correspondientes a fechas
Carpeta almacenamiento
Todos los archivo de Item Volumen a cargar se deben encontrar en la carpeta {{ folder_input_item_volumen }}
Carpeta procesados
Los archivos procesados se almacenan en la carpeta {{ folder_processed_item_volumen }}
