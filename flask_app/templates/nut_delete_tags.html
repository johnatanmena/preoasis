Desactivar/Eliminar Tags


Funcionalidad que permite desactivar o eliminar tags de la base de datos de las tablas de productos y transaccionales, normalmente este comportamiento ocurre por un procedimiento de cambio de tags, por favor no ejecutar este procedimiento sin tener un conocimiento detallado de sus consecuencias. El archivo a cargar debe ser formato xlsx, y tener las columnas: ['TAG', 'DESCRIPCION']
